## Installation

```bash
$ npm install -g node-gyp
$ npm i
```

## Prepare DB

```bash
$ npm run start:mongo
$ npm run start:init:mongo
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
